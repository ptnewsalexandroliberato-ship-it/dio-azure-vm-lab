# Lab Azure VM - DIO

## Descrição
Este repositório contém a documentação do laboratório prático de provisionamento de Máquinas Virtuais na Microsoft Azure, realizado durante o bootcamp da DIO. O objetivo foi explorar os serviços de IaaS, configurando uma VM Windows e validando o progresso no Microsoft Learn.

## Tecnologias Utilizadas
* Cloud: Microsoft Azure
* Modelo de Serviço: IaaS (Infraestrutura como Serviço)
* Sistema Operacional: Windows Server 2019 Datacenter
